Conceptos Básicos de Programación

Variables

Las variables son como cajas donde guardamos información que podemos usar después.

// Ejemplo de variables con Pokémon
let nombrePokemon = "Pikachu";
let nivelPokemon = 25;
let esEvolucionado = false;

En este ejemplo:

  1. Creamos una variable para el nombre del Pokémon
  2. Guardamos su nivel en otra variable
  3. Usamos una variable booleana para saber si está evolucionado

Vectores (Arrays)

Un vector es como una lista ordenada donde guardamos varios elementos del mismo tipo.

// Lista de Pokémon iniciales
let pokemonsIniciales = [
    "Bulbasaur",
    "Charmander",
    "Squirtle"
];

En este ejemplo:

  1. Creamos una lista llamada pokemonsIniciales
  2. Guardamos los nombres de los tres Pokémon iniciales
  3. Cada Pokémon está en una posición específica (0, 1, 2)

Bucles (Loops)

Los bucles nos permiten repetir una acción varias veces, como revisar cada Pokémon en nuestra lista.

// Revisar cada Pokémon en la lista
for (let i = 0; i < pokemonsIniciales.length; i++) {
    console.log("Pokémon número " + i + ": " + pokemonsIniciales[i]);
}

En este ejemplo:

  1. El bucle comienza en 0 (i = 0)
  2. Se repite mientras i sea menor que la cantidad de Pokémon
  3. En cada vuelta, muestra el número y nombre del Pokémon

Valor Temporal

Un valor temporal es como una caja temporal donde guardamos algo mientras hacemos un intercambio.

// Intercambiar dos Pokémon de lugar
let temp = pokemonsIniciales[0];           
pokemonsIniciales[0] = pokemonsIniciales[1]; 
pokemonsIniciales[1] = temp;

En este ejemplo:

  1. Guardamos temporalmente el primer Pokémon
  2. Movemos el segundo Pokémon al primer lugar
  3. Movemos el Pokémon guardado al segundo lugar

Condicionales (if)

Los condicionales nos permiten tomar decisiones, como verificar si un Pokémon es más fuerte que otro.

// Comparar niveles de Pokémon
let nivelPikachu = 25;
let nivelRaichu = 30;

if (nivelPikachu < nivelRaichu) {
    console.log("¡Raichu es más fuerte!");
} else {
    console.log("¡Pikachu es más fuerte!");
}

En este ejemplo:

  1. Definimos los niveles de dos Pokémon
  2. Comparamos si el nivel de Pikachu es menor que el de Raichu
  3. Mostramos un mensaje diferente según el resultado

El Método de Burbuja

El método de burbuja es un algoritmo simple que ordena elementos comparando pares adyacentes e intercambiándolos si están en el orden incorrecto.

function bubbleSort(pokemons) {
    for (let i = 0; i < pokemons.length - 1; i++) {
        for (let j = 0; j < pokemons.length - i - 1; j++) {
            if (pokemons[j].id > pokemons[j + 1].id) {
                // Intercambiar posiciones
                let temp = pokemons[j];
                pokemons[j] = pokemons[j + 1];
                pokemons[j + 1] = temp;
            }
        }
    }
}
