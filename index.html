Conceptos Básicos de Programación

Vectores (Arrays)

Un vector es una estructura que almacena múltiples elementos del mismo tipo.

let pokemons = [
    {id: 1, name: "Bulbasaur"},
    {id: 4, name: "Charmander"},
    {id: 7, name: "Squirtle"}
];

Condicionales (if)

Los condicionales nos permiten tomar decisiones en nuestro código.

if (pokemon1.id > pokemon2.id) {
    // Intercambiar posiciones
}

Bucles (for)

Los bucles nos permiten repetir acciones múltiples veces.

for (let i = 0; i < pokemons.length; i++) {
    // Procesar cada Pokémon
}

El Método de Burbuja

El método de burbuja es un algoritmo simple que ordena elementos comparando pares adyacentes e intercambiándolos si están en el orden incorrecto.

function bubbleSort(pokemons) {
    for (let i = 0; i < pokemons.length - 1; i++) {
        for (let j = 0; j < pokemons.length - i - 1; j++) {
            if (pokemons[j].id > pokemons[j + 1].id) {
                // Intercambiar posiciones
                let temp = pokemons[j];
                pokemons[j] = pokemons[j + 1];
                pokemons[j + 1] = temp;
            }
        }
    }
}
